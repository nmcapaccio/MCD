# @mihistorial/shared-types

## Propósito

Centralizar tipos TypeScript compartidos entre `apps/mobile`, `apps/web-doctor` y `supabase/functions`.

## Qué hace

- Expone tipos del dominio (paciente, consultas, documentos, tokens QR).

## Qué NO hace

- No contiene lógica de negocio.
- No contiene código de red ni integración con Supabase.
